# AI Fitness Coach

An AI-powered fitness platform that provides real-time posture correction, repetition counting, workout planning, and progress tracking using computer vision.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Express.js
- PostgreSQL
- Prisma
- FastAPI
- MediaPipe
- OpenCV
- Docker
